# 🍽️ Image-Based Calorie Estimator

## 📌 Project Overview
This project is a Machine Learning-based web application that estimates the calorie value of food items using image input.

The system predicts food type and calculates approximate calories of given weight of food.

---

## 🚀 Features
- Upload food image
- Predict food item
- Display estimated calories
- Simple and user-friendly interface

---

## 🛠️ Tech Stack
- Python
- Flask
- Machine Learning
- Deep Learning
- HTML/CSS
- SQLite

---

## 📊 Dataset
Custom food calorie dataset containing multiple Indian food items with calorie values per 100g.
